# Personal Portfolio Website

A clean, modern, fully responsive personal portfolio website built using **exclusively HTML5 and CSS3**. NO JavaScript or frameworks were used (except for the optional inline JS in the offline page button for reload).

## Project Structure
```
portfolio-website/
│
├── index.html          # Home Page
├── about.html          # About Me Page
├── skills.html         # Skills Page
├── projects.html       # Projects Showcase Page
├── contact.html        # Contact Page
├── offline.html        # Offline Fallback Page
│
├── css/
│   └── style.css       # Main Stylesheet (Variables, Flexbox, Grid)
│
├── assets/             # Images and Icons directory
│
└── manifest.json       # PWA Manifest
```

## Features
- **Responsive Design**: Mobile-first approach using Media Queries.
- **Modern Layout**: Utilizes CSS Flexbox and Grid.
- **Theme**: Professional Blue (#2563eb) and Gray color scheme.
- **PWA Ready**: Includes `manifest.json`.

## How to Run
1. Unzip the folder.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3. No server or build step required.

## Customization
- **Colors**: Edit the CSS Variables in `:root` inside `css/style.css`.
- **Content**: Edit the HTML files directly to update your information.
